# Setup
```
# Command Line
git clone https://github.com/DEV3L/making-a-minecraft-mod.git
cd making-a-minecraft-mod
./gradlew setupDecompWorkspace eclipse

# Open Eclipse
$making-a-minecraft-mod/eclipse
Run Configurations - Client
  -- GradleStart
```
